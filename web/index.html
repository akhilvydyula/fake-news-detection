Security & data

        
Design for newsrooms and compliance-minded teams: optional API keys, no training on your payloads by default, and clear separation of triage vs. ground truth.

      

      

        

          
Authentication

          
Use PLATFORM_API_KEY for a single tenant, or PLATFORM_API_KEYS (JSON map of key → org id) for multiple customers. Clients send X-API-Key; secrets never appear in JSON responses. Successful analyzes may include tenant.org_id.

        

        

          
URL fetch safety

          
URL ingestion uses size limits and blocks private/reserved hosts to reduce SSRF risk. Allowlists can be tightened per deployment.

        

        

          
Your data

          
Analysis is for real-time triage. Fine-tuning on customer data is a separate, explicit agreement — not the default path.
